Variables

Utilizamos las variables para dar algun nombre a los valores que utilizamos en nuestro proyecto. Pero estos nombres dentro de C deben seguir ciertas reglas.

El nombre de las variables deben comenzar con letras (a-z-A-Z) no se puede comenzar el nombre con números, éstos se pueden utilizar luego del primer caracter (0-9), tampoco están permitidos utilizar caracteres especiales(*,#,$,etc...) ni acentos o espacios en cualquier parte del nombre. Al final se debe colocar siempre punto y coma (;)

Declarando Variables

A continuación se mostrará algunos nombres de variables declaradas correctamente

int numero;
float decimal1;
char c;
varchar palabras_clave;

Ahora se mostrará algunos nombres de variables declaradas incorrectamente

int número;
char 1palabra;
float precio de producto;
Variables Globales

Muchas veces vamos a utilizar variables globales, ya que no solo la necesitaremos dentro de cierta funcion sino en varias. La declaramos fuera de cualquier función incluso fuera de la función principal.

Constantes

Las constantes tales como su nombre lo dice, mantiene su valor en todo el proyecto. El nombre siguen las mismas reglas que la variable y se declaran de la siguiente manera:

#define nombre valor

Algunos ejemplos:

#define pi 3,14
#define gravedad 9,8

Como podemos ver no es necesario el punto y coma al final.

Tipos de Datos

Algunos tipos de datos básicos son:

  • Números enteros definidos con la palabra clave int
  • Letras o caracteres definidos con la palabra clave char
  • Números reales o en coma flotante definidos con las palabras claves float o double
Sentencia if... else

Para tener una estructura de selección utilizamos la sentencia if... else, y lo hacemos de la siguiente manera:

if (condición){
//si se cumple la condicion se ejecuta este bloque
}else{
//sino se ejecuta este bloque
}
Sentencia while

Para tener una estructura de repetición utilizamos la sentencia while de la siguiente manera:

while (condicion){
//mientras se cumpla la condicion se ejecuta este bloque
}

De esta manera dentro del bloque debemos ir cambiando el valor de la variable de condición para que pueda salir del bloque cuando no se cumpla esa condición.

Funciones

Las funciones se utilizan para poder reutilizarlas luego en cualquier momento y no hacer el codigo redundante.

tipo_de_dato_que_retorna nombre_de_la_funcion (parametros los cuales son opcionales){
//sentencias de la funcion
return valor_con_tipo_de_dato
}

Un ejemplo de función la cual le pasamos tres notas de programación en un año y nos devuelve el promedio:

float promedio (float nota1, float nota2, float nota3){
float suma;
float promedio;
suma = nota1 + nota2 + nota3;
promedio = suma / 3;
return promedio;
}
